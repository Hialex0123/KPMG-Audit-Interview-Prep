## 二、GBA 在线测评攻略

> 来源：B站「四大摆渡人」https://www.bilibili.com/video/BV1R4sjedENg/
> 来源：知乎 https://zhuanlan.zhihu.com/p/715257617
> 来源：毕马威KPMG校招在线测评笔试真题汇总 https://www.highmarkcareer.cn/bsjq/271.html

测评分两轮，2026年新版题型已更新：

### 第一轮：认知能力测试（Cognitive Test）

| 模块 | 说明 |
|------|------|
| 译图达人 | 信封排序，越后面数量越多，可能出现根式比较，共10关 |
| 投影题 | 4×5立方体三视图，限时1分钟/题 |
| 飞行反应 | 考反应力，规则多，要盯中间飞机看 |
| 瞄准题 | 地球上插树苗，考耐心，按键有延迟 |
| 路径题 | 步数最少规划路径，总限时5分钟 |
| 逻辑推理 | 圆桌座位/集合关系等类行测题 |
| 数学推理 | 速度/行程类，类GMAT数学 |

**备考建议：**
- 提前备好草稿纸和计算器
- 新版题型持续变化，建议考前2-3天再查最新帖子
- B站"四大摆渡人"有完整视频讲解（链接见上）

### 第二轮：行为测评 + 视频面试（Video Interview）

- **英文行为面试**：用英文回答行为类问题，约6分钟
  - 真题示例：*"Please recall a time when you insisted on doing something despite the objections and misunderstandings of others..."*
- **性格测试**：选最像/最不像自己的陈述，保持真实但结合岗位特质
- **情景判断**：给出职场场景，写英文邮件或沟通方案

## 三、群面：剧本杀 Case 详解

> 来源：小红书"3天拿下KPMG offer" http://xhslink.com/o/7xTo8C8hFms
> 来源：小红书"26KPMG群面剧本杀case&单面回顾" http://xhslink.com/o/4uYMpZsrnFv
> 来源：知乎"四大群面经验|毕马威2025群面案例分析" https://zhuanlan.zhihu.com/p/26368945520

### 常见 Case 类型

| Case类型 | 说明 | 频率 |
|---------|------|------|
| **地铁线路规划** | 最高频！扮演不同角色（校长/公交会长/环保局长等），在预算和人流量约束下规划线路 | ⭐⭐⭐⭐⭐ |
| **航线规划** | 与地铁逻辑相同，换了场景 | ⭐⭐⭐ |
| **企业数字化转型危机** | 8人分角色（CEO/CMO/技术总监/投资人），解决系统故障，各有隐藏任务 | ⭐⭐⭐ |
| **企业价值观排序** | 对一组价值观或方案进行优先级排序并达成共识 | ⭐⭐ |
| **公司利润分析** | 公司利润连续下滑，分析原因提出改进措施 | ⭐⭐ |

### 地铁 Case 实操指南（最高频）

**流程：**
1. 阅读案例资料（5-15分钟，中文呈现）
2. 小组讨论（约40分钟）
3. 汇报展示（约15分钟）
4. 无需自我介绍，直接进入讨论

**高分策略：**
1. **靠白板的人先画草图**，方便汇总路线
2. **明确分工**：人流量计算、预算计算、绘制最终路线各分配一人
3. **Time Keeper**：每5分钟同步进度，体现控场意识
4. **不必抢 Leader 位**：评委更看重协作深度和创意
5. **剧本杀Case破局三步**：先抓主线矛盾 → 融入个人隐藏任务 → 适当造戏剧冲突再让步

## 四、单面：M面（经理面）高频问题

> 来源：小红书"KPMG审计面经复盘（已offer）" http://xhslink.com/o/9VZiw96Hatj
> 来源：搜狐"最新！2025年毕马威面试攻略" https://www.sohu.com/a/949834323_121956424
> 来源：牛客网毕马威校招面经 https://www.nowcoder.com/enterprise/2070/interview

### Three Why（核心必备）

| 问题 | 考察重点 | 答题方向 |
|------|---------|---------|
| Why Audit — 为什么选审计？ | 是否真的想清楚为什么入行 | 结合自身经历，体现主动思考，不背模板 |
| Why KPMG — 为什么选毕马威？ | 与公司的匹配度 | 了解毕马威特色（智能审计/ESG/大湾区业务） |
| Why You — 你的竞争优势？ | 个人背景与岗位契合 | 具体实习经历 + 技能 + 性格特质 |

### 其他高频问题

- 你对审计的理解是什么？**审计和财务有什么区别？**
- 简历中哪项经历最有挑战/最有收获？
- 遇到团队合作意见冲突你怎么办？
- 有没有考研计划？为什么选择工作？
- 你觉得自己和这份工作的匹配度体现在哪里？
- 如果再给你一次群面机会，你会改进什么？

### 面经实录（M面）

> *"面试官没有让我自我介绍，也没问各种why，而是问了我有没有考研以及为什么报那个学校，还有问我觉得自己和这份工作的匹配度。五六分钟就结束了。应该是面试官比较满意，所以没有多问。"*
> *(来源：小红书 http://xhslink.com/o/7xTo8C8hFms)*

## 五、PAR面（合伙人面）高频问题

> 来源：知乎"毕马威通关计划：笔试+M面+Par面的经验分享" https://zhuanlan.zhihu.com/p/64610956
> 来源：牛客网毕马威校招面经 https://www.nowcoder.com/enterprise/2070/interview

### 常见问题

- **英文自我介绍**（必备，建议准备1分钟版本）
- 用三个关键词描述自己
- 你的短期和长期职业规划是什么？
- 描述一次带领团队解决难题的例子（STAR法则）
- 工作中遇到挫折怎么调整？
- 四大加班很常见，你能接受吗？
- KPMG 的核心价值观是什么？你与哪些最契合？
- 毕马威和其他四大有什么区别？
- 如果同时拿到几家四大的 Offer，你如何选择？
- 对 ESG、AI 审计、数字化等行业热点的看法？

### 广州 PAR 面特点

- 氛围轻松，更像深入聊天而非"问答拷打"
- 可能会追问对**粤港澳大湾区业务**的理解
- 建议反问环节问 Partner 个人的职业成长故事

## 六、面试官最看重的特质（综合面经）

| 特质 | 说明 |
|------|------|
| 放松自然 | 多名过来人强调，紧张是最大减分项 |
| 结合经历 | 不要空讲理论，用具体案例回答（STAR法则） |
| 快速清晰 | 单面节奏快，每个问题精准简洁输出 |
| 英文表达 | 可能要求英文自我介绍，提前准备好 |
| 协作意识 | 群面时倾听他人、尊重不同意见同样重要 |
