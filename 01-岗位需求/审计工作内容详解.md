# 审计是什么？KPMG外部审计工作详解

> 📌 专为金融专业背景、无审计经验的同学整理，帮助你在面试前真正理解这个岗位。

---

## 一、审计的本质：一句话理解

> **审计 = 独立的第三方，检查一家公司的财务报告是否"说了真话"。**

公司每年要出财务报表（资产负债表、利润表、现金流量表），对外公开给投资者、银行、监管机构看。但公司自己写自己的报告，可能有造假、错误或误导的风险。

**审计师的职责**就是以独立第三方的身份，用专业方法核查这份报告——最终出具一份**审计报告**，告诉公众："这份财务报告基本上是真实公允的"或"有重大问题"。

## 二、外部审计 vs 内部审计（核心区别）

| 对比维度 | **外部审计**（KPMG做的） | **内部审计**（企业自己的） |
|---------|----------------------|------------------------|
| 服务对象 | 公司的**外部使用者**（投资者、股东、监管机构） | 公司**内部管理层** |
| 独立性 | 必须完全独立于被审计公司 | 隶属于公司本身 |
| 目的 | 对财务报表**真实性发表意见**（法定义务） | 发现内部流程漏洞、提升运营效率 |
| 法律性质 | **法定审计**，上市公司必须请外部审计师 | 非强制（部分公司设置） |
| 雇主 | 会计师事务所（四大、国内所） | 被审计公司本身 |
| 工作周期 | 每年集中忙季（1~4月年报季）+ 淡季 | 全年持续 |
| 你的角色 | KPMG审计助理 A1 | 企业内审员 |

💡 **一句话区分**：内审是公司"自查"，外审是"被查"。KPMG做的是外审，是上市公司、大型企业**必须**花钱购买的法定服务。

## 三、KPMG审计新人A1实际做什么？

### 📋 整体工作流程

```
接到项目 → 了解客户业务 → 风险识别 → 设计审计程序
→ 执行测试（收集证据）→ 编制底稿 → 出具报告
```

作为A1（新人），你主要参与**中间执行阶段**，由Senior带着做。

---

### 🔍 六大核心日常工作（真实版）

#### 1. 抽凭（最基础、最高频）
- 从客户的会计凭证中**按规则抽取样本**，拍照/扫描存档
- 核查凭证金额、日期、审批签字是否合规
- **你的感受**：枯燥但重要，考验细心，一旦出错会影响整个底稿

#### 2. 函证（发函、追函）
- 向银行、客户、供应商等第三方**发出询问函**，请他们确认余额
- 跟进回函进度，有时需亲自去银行、客户现场办理
- **你的感受**：锻炼沟通和跟进能力，了解真实商业流程

#### 3. 监盘（出差现场盘点）
- 去客户的工厂/仓库，实地**盘点存货/现金/固定资产**
- 抽样核查实物与账面是否相符
- **你的感受**：可能去制造工厂、港口、冷库——见识真实产业

#### 4. 编制审计底稿
- 用Excel/审计系统（KAT/KPMG Clara）整理证据，编制工作底稿
- 银行存款、应收账款、收入等科目
- **你的感受**：大量Excel操作，学到vlookup/数据透视表等实战技能

#### 5. 与客户沟通取证
- 向客户财务部索要资料、解释要求
- 当客户提供信息不足时，礼貌追问和催进度
- **你的感受**：锻炼职业化沟通，学会"不卑不亢"地向对方提要求

#### 6. 数据分析（KPMG特色：智能审计）
- KPMG强调数字化转型，A1也会用数据分析工具做**分析性程序**
- 例如：用Python/Excel分析客户收入趋势异常点
- **对你的优势**：你的Python/R/数据背景在这里直接有用！

### 📅 忙季 vs 淡季

| 时间 | 状态 | 说明 |
|------|------|------|
| 1月–4月 | 🔴 忙季 | 年报审计高峰，加班到晚9~11点是常态 |
| 5月–8月 | 🟡 中等 | 半年报 + 培训 + CPA备考 |
| 9月–12月 | 🟢 淡季 | 相对轻松，新人入职培训期 |

## 四、金融专业的你 vs 审计工作：有哪些相通之处？

| 你的金融背景 | 审计中的对应技能 |
|------------|---------------|
| 财务报表分析（公司金融课） | 直接用于理解客户报表、识别异常 |
| 统计/计量经济学 | 分析性程序、数据审计 |
| Python/R编程 | KPMG智能审计工具、数据提取分析 |
| 金融科技实习（监管科技、风险排查） | 与审计的"风险识别"逻辑高度相似 |
| 宏观经济理解 | 理解客户所在行业的大背景 |

💡 **关键认知**：审计不是"只有会计专业才能做"。KPMG明确不限专业，金融背景的优势在于**行业理解力 + 数据能力**，这正是智能审计时代最需要的。

## 五、外部审计的职业发展路径

```
A1 助理（0-2年）
    ↓
A2/Senior 高级助理（2-4年）—— 独立负责模块，带A1
    ↓
Manager 经理（4-7年）—— 负责整个项目，对接合伙人
    ↓
Senior Manager 高级经理
    ↓
Partner 合伙人（10年+）—— 签字合伙人，业务开拓
```

**跳出四大后的方向**（大多数人2~4年后）：
- → 上市公司财务总监/CFO团队
- → 私募基金 / 投资机构财务尽调
- → 监管机构（证监会、银保监）
- → 创业 / 自开会计师事务所

## 六、你作为金融生，面试时"为什么选审计"怎么答？

> 不要说"因为我喜欢数字"或"审计很专业"——太泛了。

**建议思路（结合你的背景）**：

> "我在广州金融科技的实习做的是监管科技和风险排查，本质上也是在用数据识别财务行为中的异常和风险点，这和审计的底层逻辑非常相通。但实习让我意识到，要真正理解一家企业的风险，需要从财务报表的源头出发——审计师接触的是最原始的财务数据，最接近企业真实运营。KPMG的智能审计方向也让我觉得，我的数据分析背景可以在这里真正发挥价值，而不是做纯粹的体力型核查工作。"
