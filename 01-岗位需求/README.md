---

## 岗位职责

1. 利用数据和分析手段，评估客户财务报告是否真实、准确并公允反映实际状况，发表专业意见
2. 参与智能化审计，包括数据分析与挖掘，推动审计流程自动化，让审计师聚焦沟通和增值服务
3. 承担客户项目的具体审计工作，包括凭证检查、账目分析、与客户沟通取证等日常核心审计环节
4. 协助团队达成项目进度要求，确保交付高质量工作成果

---

## 任职要求

- **学历**：应届本科及以上（财会、经济、管理等相关专业优先，不限专业）
- **毕业年份**：2025 或 2026 年，能按时取得学历学位证书
- **语言**：英语具备读写能力（CET-4/6、雅思、托福等证书优先）
- **能力**：较强的数字敏感度与逻辑分析能力
- **素质**：诚信正直，责任心强，学习意愿高，适应高强度快节奏环境
- **加分项**：相关审计/财务实习经历、校园领导经历、CPA/ACCA 等会计证书

## 校招流程

```
网申/简历 → 在线测评（逻辑/数理/情商） → 面试（共 2.5 小时，线下）→ Offer
```

- 面试通常包含结构化面试 + 行为面试（BEI）
- 可能涉及英文自我介绍或英文问答
